== Description ==

Simple calendar plugin.

If you have any question feel free to write comments on the plugins page: http://www.wellrootedmedia.com/life-calendar-plugin

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload plugin.
2. Activate the plugin.
3. Create a new page, and insert "[life_calendar]" in the body of the content.
4. If you want to display the time line under the calendar you can tick the checkbox under the life calendar > settings.
5. If you want to display the time line on a separate page, you can just add "[life_timeline]" to the body content.
